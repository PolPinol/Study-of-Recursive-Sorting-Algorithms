Autors: Pol Piñol Castuera (pol.pinol) | Roger Casas Aisa (roger.casas)

Llenguatge: Java
IDE: IntelliJ IDEA 2020.2.3 x64
Versió: Java 15.0.1 (onpenjdk - 15 java version "15.0.1")

Compilació: Comprovar que a 'Edit Configuration' -> 'Application' -> 'Main' -> 'VM options'
	conté '-ea' per així acceptar asserts, acte seguit 'Build Project' amb la classe 'Main'
	com a classe principal.

Execució:'Run' i per seleccionar les diferents opcions, interactuar amb la Terminal.
	Introdueix el nom del dataset que es desitja analitzar i un cop carregat 
	seleccionar les diferents opcions disponibles per a visualitzar els rankings.
	
	NOTA: Com no s'especifica a l'enunciat, assumim que totes les dades que li entra l'usuari
	seran vàlides i per tant no fem control d'excepcions.
